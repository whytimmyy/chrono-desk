# ChronoDesk

A modern desktop calendar application built using Electron.js, HTML, CSS, and JavaScript. This app allows users to view dates, navigate months, and manage daily schedules in a clean and minimal interface.

## Tech Stack

* Electron.js
* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Features

* Monthly calendar view
* Navigate between months
* Highlights current date
* Runs as a desktop application

## Getting Started

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd YOUR_REPO_NAME
npm install
npm run start
```

## Project Structure

* index.html – UI layout
* styles.css – Styling
* script.js – Calendar logic
* main.js – Electron main process
* assets/ – Static files
